# はろChat

このプロジェクトは`Node.js`を用いた軽量チャットアプリです。

`No-Redis`フォルダ内には、`Redis`を使用しない`server.js`と`package.json`が存在します。

## 環境変数

このプロジェクトを実行するには、次の環境変数を.envファイルに追加する必要があります。

* `REDIS_URL`

このプロジェクトを実行するに当たって、次の環境変数を.envファイルに追加することが推奨されています。

* `ADMIN_PASS`
* `SECRET_KEY`
